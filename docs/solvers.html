function solve!(sv::LpTimingSolver, sol::Solution)¶

solve!(sv::LpTimingSolver, sol::Solution)

Finds the optimal landing times given the ordering defined by the solution sol. The solution landing times landing and cost are modified in-place.

This function relies on solving a linear program. If the program has no solution, then earliest landing strategy is applied.
