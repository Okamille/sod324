Array¶

function shift!(v::Vector{T}, idx1::Int, idx2::Int) where {T}¶

Déplacement d’un élément dans un tableau

Paramètres

  • v – vecteur à modifier

  • idx1 – indice de l’élément à décaller

  • idx2 – indice de l’émément après déplacement

Le tableau passé en paramètre est modifié.

Renvoie

Le tableau complet

Exemple

``` v = collect(1:7)

shift!(v, 7, 1)”

=> 1×7 LinearAlgebra.Adjoint{Int64,Array{Int64,1}}:

7 1 2 3 4 5 6 ```

Voir aussi

  • permutate()

  • circshift()

function new_model(; solver = :auto, mode = :lp, log_level = 0)¶

Création d’un modèle de solver externe (:cplex, :glpk, …)

Paramètres

  • solver –

    symbole :cplex, :glpk, :clp, :cbc

    • :cbc : permet de faire du MIP

    • :clp : resteint aux problème LP mais plus efficace de cbc pour les relaxations.

  • mode – (:lp ou :mip) utile pour choisir entre :clp ou :cbc

  • log_level (expérimental) – entier 0 pour silence, sinon 1, 2, …

Autre solver à supporter plus tard Gurobi (concurrent de cplex)

Attention

DEPENDANCE EXTERNE : module Args (pour Args.get(« external_mip_solver »))
